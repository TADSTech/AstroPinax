Set as Default New Tab

Chromium browsers (Chrome, Edge, Brave, Opera) don't let sites override the new tab page directly — you route it through a lightweight "custom new tab" extension.

  1. Open your browser's extension store and install a “Custom New Tab URL” extension (search that exact phrase).
  2. Open the extension's options and paste the AstroPinax URL below into the new tab field.
  3. Save. Every new tab now opens AstroPinax.
